# Changelog

All notable changes to this project will be documented in this file.

## [Unreleased]

### Added
- Initial syntax highlighting for SudoLang files (.sudo, .sudo.md, .mdc)
- Support for all SudoLang language constructs (keywords, strings, variables, comments, functions, interfaces, commands, operators)
- Markdown integration for mixed-format files
- Context-aware indentation
- Support for line (//) and block (/* */) comments
- Filetype detection for .sudo, .sudo.md, and .mdc extensions
- Zero-config setup with optional configuration
- Automated tests and CI/CD pipeline
- Comprehensive documentation
